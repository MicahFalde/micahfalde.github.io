The church management system is the data backbone — people, giving, check-in, attendance. Our #1 criterion (R1) is a great two-way API so we can build our own tools on top. Two real surprises shrank the field: ChurchTrac has no API, and "Breeze" is now "Tithely Church Management."

How to choose: if you want to move fast with minimal ops, building on Planning Center's free API lets you customize at the integration layer (your own dashboards, kiosks, attendance tools) without owning a server or security patching. If you want full data ownership and have a developer, a self-hosted base (ChurchCRM — MIT, or Rock) is the path — accept owning the server and patching. ChurchCRM is also worth standing up as a parallel data-sovereignty experiment. Full detail in docs/chms-comparison.md.

No free tool has frictionless live SongSelect import — that's a paid-product feature. FreeShow is the best free PCO+streaming combo (manage lyrics manually); OpenLP is the other free option that talks to PCO.

CCLI is three separate purchases — the licenses are the hard legal requirement (needed the moment you project or print copyrighted lyrics); the software is optional. Licenses are priced by average weekly attendance.

① Copyright License

~$246/yr

The baseline. Project & print lyrics, custom songbooks/arrangements for in-person worship. Unlocks SongSelect. Does not cover online/streaming.

② Streaming License

~$175/yr

Required the moment a worship song appears in a livestream or posted video. Cheaper (~$84) if you already hold the Copyright License. "Plus" adds backing-track rights.

③ SongSelect

~$171/yr

The catalog (230k+ songs) — lyrics, chord charts, lead sheets, transposition. A subscription, not a license; requires the Copyright License. Advanced ~$171 / Premium ~$215.

~$592/yr is the fixed legal floor (Copyright + Streaming + SongSelect) regardless of which software you pick. You must also report song usage — ProPresenter and Planning Center both auto-report (PCO every Monday); OpenLP does not.

How the pieces wire together

Planning Center→ run-of-show →ProPresenter / FreeShow / OpenLP
ProPresenter or PCO→ usage report →CCLI

Pick your size and what you need — see the cost live. Figures are current-approximate annual costs; card-processing fees on giving (~2–3%) are separate, and self-host options don't price the developer's time.

Three complete bundles, each suited to a different way of operating — pick by what fits how you want to run things, not by a ranking. The ~$592/yr CCLI floor is fixed in all three; the real choice is the software on top.
